This program converts a vtkPolyData image into volume representation (vtkImageData) where the foreground voxels are 1 and the background voxels are 0. Internally vtkPolyDataToImageStencil is utilized. The resultant image is saved to disk in a metaimage file format. vtkMarchingCubes is applied to this file format, and a vtk file is outputted.
